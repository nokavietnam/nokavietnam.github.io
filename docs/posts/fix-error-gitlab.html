    # Fix lỗi permission push gitlab

    Generate SSH key pair

    Mở gitbash và run lệnh bên dưới để sinh cặp key

    ssh-keygen -t rsa -b 2048 -C "<comment>"
    1

    Nhập file chứa cặp key

    C:/Users/{yourDesktopName}/.ssh/id_rsa
    1

    Phần nhập passphrase các bạn để trống

    Phần nhập lại passphrase các bạn cũng để trống

    Sau khi gen xong các bạn vào thư mục "C:/Users/{yourDesktopName}/.ssh/" sau đó các bạn copy nội dung file id_rsa.pub và thêm vào gitlab

    Xem thêm ở đây (opens new window)

    Các bạn bổ sung config cho git

    Kiểm tra lại key bạn add

    #
eval `ssh-agent -s`

# add key
ssh-add C:/Users/{yourDesktopName}/.ssh/id_rsa

# Check
ssh -T git@gitlab.com

Welcom to GitLab, @{username của bạn} => nếu thấy dòng này là ok
