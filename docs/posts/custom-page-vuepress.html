# Tạo sidebar cho bài viết

Các bạn mở file config.js. Trong themeConfig các bạn bổ sung thêm sidebar: 'auto'

module.exports = {
  ...
  themeConfig: {
    ...
    sidebar: 'auto',
  }

}

# Tạo trang đường dẫn next và prev page

Nếu bạn có nhiều bài viết trong cùng một serial và bạn muốn hiển thị bài viết trước vá sau của serial đó.

Để config page preview và next page. Bạn chỉ bổ sung 2 config prev và next ở đầu trang.

prev: đường dẫn đến trang trước đó.

next: đướng dẫn đến trang sau.

---
...
prev: /posts/abc.html
next: /posts/xyz.html
---

# Cài đặt plugin back to top

Bổ sung config plugin trong file config.js

module.exports = {
  plugins: ['@vuepress/back-to-top']
}

# Tùy chỉnh size hình ảnh trong markdown file

Bổ sung config

module.exports = {
  ...
  markdown: {
    extendMarkdown: (md) => {
      md.use(require('markdown-it-imsize'));
    }
  }
}
