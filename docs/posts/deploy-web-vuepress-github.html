      # Hướng dẫn tạo tạo trang web đơn giản với Github

      Ở bài trước mình đã hướng dẫn các bạn tạo một web đơn giản bằng VuePress.

      Nhưng chỉ có thể run ở local.

      Bài này mình sẽ hướng dẫn các bạn deploy source code lên github và tạo link để mọi người có thể truy cập vô blog của các bạn.

      # Build source

      Để build source bạn mở command hoặc terminal và chạy lệnh sau

      Sau khi run thì các tập tin kết quả sẽ nằm trong thư mục .vuepress/dist. Nhưng để có thể deploy lên Github Page chúng ta cần chỉnh lại một chút. Bạn mở file .vuepress/config.js và thêm => dest: 'docs'

















       


      module.exports = {
  themeConfig: {
    title: 'Tên blog',
    description: 'Mô tả về blog',
    // Thiết lập các liên kế trên navbar
    nav: [
      { text: 'Trang chủ', link: '/' },
      {
        text: 'Chuyên mục',
        items: [
          { text: 'C#', link: '/csharp' },
          { text: 'Java', link: '/java' },
        ],
      },
    ],
  },
  dest: 'docs',
};

      Chúng ta chạy lại lệnh trên

      Lúc này chúng ta sẽ nhìn thấy các file html, css, js,... build ra sẽ nằm trong thư mục docs.

      Xong phần build chúng sẽ sang bước thứ 2 là tạo và push lên github

      # Push code lên github

      # Bước 1

      Bạn cần tạo tài khoản Github cái này các bạn tự search Google

      # Bước 2

      Các bạn tạo 1 repository trùng với username Github của các bạn cộng với ".github.io".

      Ví dụ: username là abc123 thì mình sẽ tạo repository abc123.github.io => chú ý chỗ này cần chính xác

      Để biết được username Github của bạn là gì bạn chọn vào avatar bên phải trên cùng và chọn vào your profile, xong bạn nhìn lên trên thanh địa chỉ sau url https://github.com/ là username của bạn.

      Ví dụ: https://github.com/abc123 => abc123 là username

      Các bạn chọn Public

      Tiếp theo trong phần Initialize this repository with các bạn chú ý bỏ tick tất cả

      Sau khi tạo xong các bạn sẽ tới trang hướng dẫn của github tương tự như bên đưới

      Các bạn chạy mở source vuepress của chúng ta lên và chạy các lệnh như trong github của các bạn hướng dẫn.

      Chú ý bỏ dòng đầu tiên

       







      echo "# structual-nextjs" >> README.md ==> không chạy dòng này

      # Vô Github

      Sau khi push source lên Github các bạn đường dẫn Github của các bạn

      Như ví dụ trên thì mình sẽ đường dẫn https://github.com/nokavietnam/abc123.github.io.git

      # Setting

      Tiếp theo các bạn chọn qua tab Settings

      Và chọn Pages

      Ở phần Source các bạn click vào combobox và chọn /docs

      Ấn save

      Done!

      Các bạn vô đường dẫn https://{username}.github.io bạn sẽ thấy của web của các bạn!
